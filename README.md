# Civio Graphs Public

📊 Open-source data visualizations by [Civio](https://civio.es/).

## Directory

| Chart | Description | Languages | Stack | Published |
|-------|-------------|-----------|-------|-----------|
| [Isolation Sanction Reasons](justicia/aislamiento-prisiones/motivos-sanciones) | Interactive square-packing visualization showing prison isolation sanctions by disciplinary reason code (2020-2025). | [Sanctions by Prison Administration](justicia/aislamiento-prisiones/sanciones-administracion) | Stacked bar chart comparing solitary confinement sanctions and protective isolation across Spain's three prison administrations (AGE, Catalonia, Basque Country) over time. | Spanish | Svelte 5, Vite 7, D3 | March 2026 |
| [Time in Limbo](justicia/aislamiento-prisiones/tiempos-limbo) | Canvas-based waffle chart visualizing how long inmates spend in isolation in Spanish prisons.
